# DayDayUp — 新概念英语在线学习平台

基于 Cloudflare 全栈部署的新概念英语学习网站。

## 功能
- 📖 课文阅读（中英对照）
- 🔊 音频播放（支持调速）
- 📝 单词卡片练习
- ✅ 课后练习题
- 📊 本地学习进度追踪

## 技术栈
- 前端：React 18 + Vite + Tailwind CSS
- 后端：Hono.js on Cloudflare Workers
- 数据库：Cloudflare D1
- 存储：Cloudflare R2
- 部署：Cloudflare Pages

## 快速开始

```bash
npm install
npm run dev
```

## 部署

```bash
npm run deploy
```
